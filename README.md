Repositori ini digunakan untuk github page Physicsdigle. Harapannya bisa digunakan untuk belajar Fisika melalui simulasi sehingga lebih representatif.
Bagi kalian yang ingin berkontribusi pada page tersebut saya persilakan.

Saya berikan keterangan mengenai struktur repositori ini.


_includes  : merupakan sidebar yang digunakan untuk menambah menu daftar dan menautkan tautan halaman.

_layouts   : merupakan elemen tampilan page Physicsdigle.

assets      : merupakan lokasi tempat estetika (assets/css) dan lokasi logika simulasi (assets/js/simulations).

topics      : merupakan lokasi halaman html simulasi.
